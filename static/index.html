Social Media app

About

This is a Python Flask app hosted on pythonanywhere.com. It functions as an API for selected FB post data.
The data were extracted from the Facebook Graph API and stored in a SQLite3 database.

Endpoints

  • Summary
  • Posts
  • Types
  • Posts - filter by type
  • Posts - filter by date range with ?since=YYYY-MM-DD&until=YYYY-MM-DD
  • Group by day and filter by date range with ?grouping=day
